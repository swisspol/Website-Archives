Description :

MetaFile Editor is a utility that allows to see and edit the content of an Infinity MetaFile, the file format for the Infinity 3D engine.

MetaFile Editor permits you to preview the content of texture images embended in the MetaFile and add / edit many types of tags: image, sound, copyright, description...
For example, you can use it to edit the data required by the game WaterRace, like the physic parameters of the ships of the game.

The MetaFile file format is supported by all French Touch tools. This file format brings many new improvements over the old object file format including optimized data organization and compressed textures.

 

Software and hardware requirements :

Hardware requirements:

Any PowerMacintosh.

Software requirements:

MetaFile Editor requires MacOS 7.1 or later. MacOS 8.5 is highly recommended.
MetaFile Editor requires QuickTime™ and the Appearance Manager.
