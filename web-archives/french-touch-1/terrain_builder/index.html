Description :

Terrain Builder is a powerful 3D world editor designed to create and edit levels (or terrains) for French Touch games with real-time rendering through the Infinity 3D engine.

You can use to modify current WaterRace levels or create totally amazing levels from the ground up.

Terrain Builder is the best 3D level editing tool for the Mac. Check the screenshots section to learn more.

Terrain Builder creates new terrains using gray levels images. You can then apply textures and lighting on these terrains, add objects, place cameras, create animations… and most of all, Terrain Builder is incredibly easy to use.

 

Software and hardware requirements :

Hardware requirements:

Minimal configuration: Power Macintosh with a 832x624 screen, but the Rave software engine provided by QuickDraw 3D™ Rave will probably be too slow for real-time editing.

Best configuration: a G3 (all G3 Power Macintoshes have ATI 3D chips on board) with 4Mb of VRAM or a PowerMacintosh with a 3D accelerator card (with at least 4Mb of VRAM) which is Rave compliant, and a 1024x768 screen.

Software requirements:

Terrain Builder requires MacOS 7.1 or later. MacOS 8.5 is highly recommended.
Terrain Builder requires QuickTime™ and the Appearance Manager.
Terrain Builder does not require QuickDraw 3D™ to run, but only Rave: the “QuickDraw™ 3D RAVE” extension must be present in the extensions folder of the System folder.
