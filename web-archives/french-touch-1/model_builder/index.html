Description :

Model Builder is a free powerful 3D editor designed to create and edit object files with real-time rendering through the Infinity 3D Rave engine. Model Builder can import the meshes contained in 3DMF files, DXF files or OBJ files, modify them (position, rotation, scale...), add colors or textures, and save the result as a standard Infinity MetaFile.

Model Builder was developed totally from the ground up as a real-time editor and is based on the Infinity 3D engine. As a consequence, it is the fastest 3D tool available on the Mac platform even on low-end PowerMacs.

Powerful features include:

  • high-precision (1/100 of second) sequencer with linear or quaternion interpolation,
  • linking,
  • skeleton-based mesh deformation,
  • hi-quality QuickTime™ compressed textures,
  • texture transparency,
  • vertex-based transparency,
  • Gouraud, Lambert and pseudo-Phong shading,
  • support for multiple audio tracks,
  • a vertex per vertex texture mapping editor,
  • rotoscoping,
  • multiple texture sets,
  • cameras,
  • visual FX,
  • particle systems,
  • custom audio engine,
  • and much more...

Model Builder is not a 3D modeler: it does not allow you to create meshes.

Model Builder allows you to edit and modify any 3D model included with the game WaterRace.
You can also use Model Builder to create 3D models you can include in the terrains you build with Infinity Terrain Builder, the 3D terrain editor available for free from French Touch.

 

Software and hardware requirements :

Hardware requirements:

Minimal configuration: Power Macintosh with a 832x624 screen, but the Rave software engine provided by QuickDraw 3D™ Rave will probably be too slow for real-time editing.

Best configuration: a G3 (all G3 Power Macintoshes have ATI 3D chips on board) with 4Mb of VRAM or a PowerMacintosh PCI with a 3D accelerator card (with at least 4Mb of VRAM) which is Rave compliant, and a 1024x768 screen.

Software requirements:

Model Builder requires MacOS 7.1 or later. MacOS 8.5 is highly recommended.
Model Builder QuickTime™ and the Appearance Manager.
Model Builder does not require QuickDraw 3D™ to run, but only Rave: the “QuickDraw™ 3D RAVE” extension must be present in the extensions folder of the System folder.

Check out Infinity Terrain Builder :

Infinity Terrain Builder is a free real-time 3D terrain editor for French Touch games. It allows you to use in custom terrains the 3D models you have created with Model Builder.

Read the complete description and download it on the Infinity Terrain Builder web page.

 

Check out Infinity D-Fusion :

Infinity D-Fusion is a stand alone player for 3D model animations stored in Infinity MetaFiles. It is less than 150Kb but includes a complete version of the Infinity 3D Engine, allowing you to play full screen the animations contained in any Infinity MetaFiles.

Create awesome 3D real-time animation using Model Builder, then just distribute them over the Internet with the Infinity D-Fusion player!

Read the complete description and download it on the Infinity D-Fusion web page.

 

Check out Chronos :

Chronos is a free screen saver that uses 3D model animations stored in Infinity MetaFiles as modules. It includes a complete version of the Infinity 3D Engine and is able to play full screen the 3D animations when you do not use your Mac.

Chronos is the first real-time 3D screen saver for the Mac.

Read the complete description and download it with its animated modules on the Chronos web page.

 

Check out Infinity Rasterizer :
Infinity Rasterizer is a free utility that renders hi-resolution screenshots of 3D models stored in MetaFiles (up to 16000x16000 pixels provided Infinity Rasterizer has enough free memory).
Now available as a Photoshop™ 5.x plug-in.

Read the complete description and download it on the Infinity Rasterizer web page.
