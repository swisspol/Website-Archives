Persistence Web - Starting at USD $1.99 per month

Persistence Web is a special version of Persistence that comes with professional-grade online data storage. You get the software at no charge, but instead pay monthly based on the amount of backup data you have.

The online storage is provided through Amazon Web Services, using the same data storage infrastructure that Amazon uses to run its own global network of web sites. The subscription process and billing is easy, secure and handled completely by Amazon Payments.

This high-quality online storage is offered at a very affordable prices: for instance, only USD $0.16 / Gb / month for 50 Gb of backup data, while most online backup companies charge up to USD $1.00 / Gb / month, or even a lot more if you're a business and not an individual. This great deal combined with the powerful features of Persistence, makes it the ideal online backup solution for Mac OS X.

This version is best for all users (novice or experts) who want a quality, immediately up-and-running, no-maintenance online backup experience.
Product specific features:
  • All the great features of Persistence, with the restriction that Persistence Web can only backup your data to the Amazon Web Services data centers and requires an active subscription to the Persistence Web product at Amazon Payments
  • Comes with professional-grade online storage provided by Amazon Web Services
  • You can choose to store your backup data in the United States or in Europe, whichever is closest to your location
  • No setup or cancellation fees
  • Subscribe / unsubscribe at any time
  • Monthly payment charged directly to your credit card by Amazon Payments

Subscription prices:
  • Up to 10 Gb of backup data (typically enough for all your documents and email, as well as a small iPhoto library):
    Only USD $1.99 / month
  • Between 10 Gb and 50 Gb of backup data (about 6,500 audio tracks or 10,000 pictures, typically enough for both a large size iPhoto library and a large size iTunes library):
    Only USD $7.99 / month
  • Between 50 Gb and 100 Gb of backup data (about 13,000 audio tracks or 20,000 pictures):
    Only USD $14.99 / month
  • More than 100 Gb of backup data:
    USD $0.18 / Gb / month
  • If you transfer more than 100 Gb of backup data per month, for each extra Gb transferred you are charged:
    USD $0.15
The prices above are for backup data stored in the United States. If you choose to store your backup data in Europe, then subscriptions are slightly more expensive. All the pricing details are available on the Persistence Web product page at Amazon Payments.
Buying and activating Persistence Web:
  1. Go to the Persistence Web product page at Amazon Payments to create your account (if you don't already have one) and subscribe to Persistence Web.
  2. Once you have completed the subscription, Amazon Payments will immediately provide you an activation key.
  3. Download the latest version of Persistence Web using the link on the right side of the page, launch it and enter the activation key.

If you want to later manage your Persistence Web subscription or review the transaction history, simply visit Persistence Web billing page at Amazon Payments. If you have questions regarding Persistence Web billing, send an email to:
application-payments@amazon.com.
