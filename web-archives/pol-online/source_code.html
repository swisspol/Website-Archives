Source code for developers:

This page contains source code examples that are available for free. You can use them for whatever you want.
Note that most of this source code is not carbonized and therefore won't run on OS X.

To download the Stuff-It 5.0 archive containing the desired source code, simply click on the related file icon.

Important: this source code is provided "AS IS" without any explicit or implicite guarantee.
Also note that I unfortunately do not have time to answers e-mails with questions about this source code.

 

Hold-Up Source Code2003-07-01
The full source code under GPL of the innovative Mac only accounting program that got great reviews from the european Mac web sites and magazines.

System requirements: Mac OS 8.6 or Mac OS X 10.1

 

WaterRace Source Code2003-01-13
Yes, that's right! The full-source code of the 3D boat racing game released by French Touch early 2001.

Everything's there: complete 3D engine based on former 3D API Rave, physic engine, network code, AI code, many code snippets... a consequent amount of source code!

This source code is copyright French Touch and available under the terms of the GPL license.

System requirements: MacOS 9, Code Warrior Pro

 

Render In Texture2002-09-05
OpenGL / Cocoa demo program for rendering into a texture using new API of MacOS X 10.2.

System requirements: MacOS X 10.2

 

iTalk Source Code2001-10-19
Source code of my abandonned "iTalk" project (refer to the "Vintage" area for more information).

This source code contains snippets for:
• TCP/IP connections with OpenTransport,
• low-level audio recording and playback with the Sound Manager,
• mu-law audio compression,
• high-pass filtering.

System requirements: MacOS 8 or 9

 

Dialog Controls2001-07-29
This sample code defines several Appareance compliant custom controls you can use easily re-use in your dialogs: a color picker, a pict control (with 3 modes: scale, clip or live drag), a text control...

System requirements: MacOS 8 or 9

 

Folder Utilities2001-07-29
Some utility routines to find a folder by its name inside another folder (also works with aliases), see if a folder exist or get the name of a folder from its vRefNum and ID.

System requirements: MacOS 8 or 9

 

NewList Manager2001-07-29
This sample code defines a new list manager to replace the one provided with the toolbox. It uses the exact same API but provides: custom look, live dragging, etc... Easy to customize for your need.

System requirements: MacOS 8 or 9

 

Audio Engine2001-02-18
This sample code contains a complete 2D/3D audio engine with the following features:
• up to 128 simultaneous channels with independant pitch, panoramic and volume controls,
• all mixing done in floating-point computations for highest possible quality,
• linear interpolation,
• 16 Bits 22Khz mono or stereo input,
• 16 Bits 44Khz interpolated stereo output,
• stereo localization,
• low latency,
• looping sounds,
• random sounds,
• can be connected to multiple applications at the same time.

System requirements: MacOS 8 or 9

 

Navigation Services2001-01-13
This sample code allows to ask the user for a file or ask the user where to save a file using the Navigation Services.
This sample code is 100% carbonized and ready for MacOS X.

System requirements: MacOS 8 or 9, MacOS X

 

Audio CD2000-11-11
This sample code allows you to play audio CDs from your program. It contains all the necessary routines to manipulate the CD driver: play and loop a given track, set / get volume, count tracks, check for inserted CD...
Works with official Apple CD/DVD players only.

System requirements: MacOS 8 or 9

 

GetKeys Macros2000-11-11
A set of constants and macros to use easily GetKeys().

System requirements: MacOS 8 or 9

 

MenuBar2000-11-11
This sample code allows you to hide or show the menubar and the control strip (if present) to display stuff really full-screen.

System requirements: MacOS 8 or 9

 

MP3 Player2000-11-11
This sample code allows you to play and loop indefinitely any MP3 file (no VBR though) truly asynchronously. It does not use QuickTime to decompress the MP3 data, but low-level Sound Manager interface; QuickTime is only used to extract the decompression info from the MP3 file.
Since this code uses the Sound Manager conversion routines to decompress the MP3 data, which code is already licensed, you don't need to pay any royalties to use this sample code.

System requirements: MacOS 8 or 9

 

QDesign Player2000-11-11
This sample code allows you to play and loop indefinitely any QDesign encoded AIFF file truly asynchronously. It does not use QuickTime to decompress the QDesign data, but low-level Sound Manager interface.

System requirements: MacOS 8 or 9

 

Screen Capture2000-11-11
This sample code allows you take screenshots of whatever area of the screen and output the result in a standard PICT file.

System requirements: MacOS 8 or 9

 

ScreenLib2000-11-11
This sample code, based on the Display Manager, allows you to manipulate screen devices: change resolution, depth, pick up a screen in multi-monitors environnements, prompt the user for the refresh rate, and more...

System requirements: MacOS 8 or 9

 

Sound Utilities2000-11-11
This sample code contains a set of useful routines to load and play MacOS sounds in standard System 7 sound format ('snd ' resources and 'sfil' files).

System requirements: MacOS 8 or 9

 

Spinning Cursor2000-11-11
This sample code allows you replace the mouse with the usual spinning cursor. The cursor spinning animation is totally asynchronous.

System requirements: MacOS 8 or 9
