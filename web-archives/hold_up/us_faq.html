F.A.Q. list

Nothing is displayed in the transaction list or I just created a transaction but it does not appear in the list?

    Make sure the find field is empty.

The account balance and the sum displayed under the transaction list do not match?

    The sum displayed under the transaction lists takes only into account visible transactions so if you have selected a filter or if the search field is not empty, it may differ from the account balance.

How should I specify the initial balance of an account?

    Simply create a transaction whose amount is equal to the initial balance of the account.

Can Hold-Up handle other currencies than the US dollar?

    You can set an account to use any possible currency: go to the account info dialog to define the currency of the account.
    You may also enter transaction amounts in another currency than the currency of the account thanks to the currency conversion feature (consult the users manual for more info).

Can Hold-Up create transfers from one account to another?

    Hold-Up has no such feature implemented yet.
    However you can create transfers manually: create a transaction in one account, then copy-paste it in the other account and inverse its Expense checkbox state.
    You may even create scheduled transfers by creating two identical scheduled transactions with opposite amounts.

I already own a Basic version. Can I update to a Pro version at a lower price?

    Definitely. On the online ordering page, you may buy an update to the Pro version at a low price.

Where are located the data files of Hold-Up?

    The data files of Hold-Up are located by default inside the "Hold-Up accounts" folder located inside your "Documents" folder.

I've moved the "Hold-Up accounts" folder from one computer to another, but I've lost the categories and payment types. What happened?

    You also need to copy the "Hold-Up preferences" file located into the "Preferences" folder to the "Preferences" folder of the other computer.

If I use the MacOS 8/9 version of Hold-Up, will the file still be recognized by the MacOS X version?

    Yes. All versions of Hold-Up (Basic, Pro, MacOS 9, MacOS X...) use the exact same file format.

I can't import data into Hold-Up. The files I want to import do not appear in the Open dialog box.

    Make sure the files you want to import have the 'TEXT' type.

I try to import a QIF file, but nothing happens?

    Your QIF may contain only information Hold-Up is not able to import or it is not correctly written.
    If it does not contain any "!Type" tag, try inserting the "!Type:Cash" tag followed by a return at the beginning of the file.
