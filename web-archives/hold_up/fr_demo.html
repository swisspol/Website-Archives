Téléchargement de la version Démo de Hold-Up

Nous vous invitons à télécharger la version de démonstration de Hold-Up: cette version est illimitée dans le temps mais n'est pas complètement fonctionnelle.
Vous pouvez consultez la configuration requise pour faire fonctionner la démo ici.

Le version de démonstration vous permet d'évaluer le produit et de voir si vous souhaitez acheter la version complète.
 


Téléchargement de la Démo

Cliquez sur l'une des images ci-dessous pour télécharger le fichier dans la langue de votre choix (environ 1,4 Mo). Ce fichier contient la version de démonstation de Hold-Up, des comptes d'exemple ainsi que le manuel détaillé (le manuel n'est pas disponible en langue italienne).

Vous aurez besoin de Aladdin Stuff-It Expander 5.0 pour décompacter le fichier.
