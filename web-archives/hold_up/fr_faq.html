Liste des questions

Rien n'est affiché dans la liste des opérations ou je viens d'entrer une opération mais elle n'apparaît pas?

    Assurez-vous que la case de recherche ne contient aucun texte.

Le solde et la somme affichée en bas de la liste de correspondent pas?

    La somme affichée en bas de la liste ne tient compte uniquement des opérations visibles. Ainsi, si vous avez sélectionné un filtre or si la case de recherche n'est pas vide, elle peut différer du solde du compte.

Comment, lors d'une première utilisation, indique-t'on le solde du compte?

    Il suffit de créer une opération normale avec le solde initial du compte comme montant.

Est-ce que Hold-Up peut gérer des comptes dans d'autres monnaies que le Franc Français?

    Un compte peut utiliser n'importe quelle monnaie: allez dans la boîte de dialogue des infos du compte pour définir la monnaie utilisée.
    Vous pouvez même entrer des montants d'opérations dans une autre monnaie que celle du compte grâce à la fonction de conversion (consultez le manuel pour plus d'information).

J'ai actuellement toute ma compta personnelle sur Tous Comptes Faits. Est-il possible d'importer ces données dans Hold Up?

    Oui, après les avoir exportés en format Texte tabulé depuis Tous Comptes Faits. Cela dit, toutes les données ne sont pas importées comme les sous-catégories par exemple.
    Le mieux est que vous essayiez avec la version de démonstration de Hold-Up.

Est-il possible de faire un transfert d'un compte à un autre?

    Il n'y a pas de fonction prévue pour cela dans Hold-Up pour l'instant.
    Par contre, on peut le faire facilement "à la main": on crée l'opération dans un compte puis on la copie-colle dans un autre et on inverse l'état de la case Dépense.
    Vous pouvez même créer des virements automatiques en créant 2 opérations automatiques identiques sauf pour le signe du montant.

Je possède déjà une version Basique: puis-je acquérir une version Pro à prix réduit?

    Tout à fait. Sur la page de commande en ligne, vous pouvez acheter une mise à jour de la version Basique vers la version Pro à un tarif réduit.

Où sont enregistrés les fichiers de données de Hold-Up?

    Les fichiers de Hold-Up sont stockés par défaut dans le dossier "Comptes Hold-Up" situé dans votre dossier "Documents".

J'ai déplacé le dossier "Comptes Hold-Up" d'un ordinateur à l'autre et j'ai perdu mes catégories et modes de paiement. Que faire?

    Vous devez également copier le fichier "Préférences de Hold-Up" situé dans le dossier "Préférences" vers le dossier "Préférences" de l'autre ordinateur.

Si je travaille sur la version MacOS 8/9 de Hold-Up, est-ce que les fichiers seront toujours reconnus par la version MacOS X?

    Oui. Toutes les versions de Hold-Up (Basique, Pro, MacOS 9, MacOS X...) utilisent exactement le même format de fichier.

Je ne peux pas importer mes fichiers de données dans Hold-Up: ils n'apparaissent pas dans la boîte de dialogue d'ouverture de fichiers.

    Assurez-vous que les fichiers que vous souhaitez importer sont de type 'TEXT'.

J'essai d'importer un fichier QIF, mais rien ne se passe?

    Votre fichier QIF peut ne contenir que des informations que Hold-Up ne peut importer ou il n'est pas écrit correctement.
    S'il ne contient aucun tag "!Type", essayez d'insérer le tag "!Type:Cash" suivi d'un retour à la ligne au début du fichier.
