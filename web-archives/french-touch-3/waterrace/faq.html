Product :

1) When will WaterRace be out ?

We're planning to release the game before the summer 99.

 

2) How will it be distributed?

So far we plan to distribute WaterRace as a shareware over the Internet. That's why we are designing the game so it fits in a few Mb archive.
Of course if we find a publisher for the game we might consider going commercial.

 

3) Will I be able to create my own boats or circuits ?

Sure, thanks to Fusion and WaterRace Terrain Builder.

 

4) What does DR stand for? ?

DR means "Developer Release" i.e. this version of the game is for internal use only. On the contrary, the suffix "Beta" means this version will be sent to our beta-testers.

 

Performances :

1) What are the actual performances of the game ?

So far the DR3 runs at 20 FPS in 640x480 millions of color on an IMac-B (with Rage Pro 6Mb) with full visual effects.

 

2) What is the required configuration to run WaterRace?

Any G3 computer or PowerPC computer with a video board that includes a Rave compatible 3D chip.
We are still working on a software version of Infinity so it works on any PowerMac.
