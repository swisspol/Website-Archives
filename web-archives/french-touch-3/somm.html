> Brand new softwares!

French Touch is proud to release 3 new products all based on the Infinity 3D Engine.

Fusion 2.5 is a must-have update of Fusion: improved interface, improved caracter animation, and rotoscoping are among the new features.

D-Fusion 1.0 is a stand alone player for Fusion object files. It allows you to play full screen the animations contained in Fusion object files.

Chronos 1.0 is a screen saver that uses animations included in Fusion object files as modules.
Chronos is the first real-time 3D screen saver for the Mac.

And all of them are FREEWARE!
The conference title is "L’univers des jeux" and it takes place on Saturday 18th September from 10:00 to 12:00, Salle Sirius, hall 7-3, Porte de Versailles, Paris.
