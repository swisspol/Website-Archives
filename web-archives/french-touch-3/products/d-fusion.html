What is D-Fusion!

D-Fusion is a stand alone player for Fusion object files. It is less than 150Kb but includes both the Infinity 3D Rave Engine and the Fusion sequencer allowing you to play full screen the animations contained in Fusion object files by just dropping the object file onto D-Fusion's icon or double-clicking onto the D-Fusion's icon.
And most of all, D-Fusion is freeware!

You can directly use the Fusion object files in D-Fusion: no conversion process is required.

Create awesome 3D real-time animation using Fusion, then just distribute them over the Internet with the D-Fusion player!

 

System requirements :

To be able to use D-Fusion, you need:
• a PowerPC processor MacOS system - a G3 (all G3 Power Macintoshes have ATI 3D chips on board) or a PowerMacintosh with a 3D accelerator card is highly recommended.
• System 7.1 or later.
• QuickDraw 3D Rave.
