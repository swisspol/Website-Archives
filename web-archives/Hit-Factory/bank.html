Les fichiers Bank:


 Les banks sont un format de fichier son spécifique à Hit Factory; il s'agit simplement d'un fichier contenant plusieurs samples. Vous ne pouvez les utiliser qu'avec le sampler. Le nom "Bank" est écrit sur leur icône. Si vous avez chargé un tel fichier sur une de ses piste, deux nouveaux boutons apparaîssent en haut de celle-ci. Cliquez dessus afin de sélectionner le sample que vous souhaitez entendre.

 Vous pouvez créer vous même facilement de tels fichiers en utilisant le logiciel Bank Factory, fourni avec Hit Factory. L'intérêt des banks est de pouvoir "jongler" entre plusieurs samples sur une même piste et d'augmenter ainsi virtuellement ainsi le nombre de pistes du sampler.

Description de Bank Factory:




 Ce petit utilitaire va vous permettre de créer des fichiers Bank afin de les utiliser avec le logiciel Hit Factory.
Bank Factory est en version FAT, ce qui signifie qu'il fonctionne sur les Mac 68K et qu'il est optimisé pour PowerMacintosh ou compatible pour fonctionner. Le programme nécessite 500K de mémoire vive pour fonctionner.

Nouveautés de la version 2.1:
  • Possibilité d'ajouter un texte, qui apparaîtra dans la fenêtre "Lire les informations" sous le Finder (bouton "Bank infos").
  • Support des bulles d'aide.
  • Quelques corrections.
Vous trouverez la version 2.1 de Bank Factory ainsi que son mode d'emploi à la rubrique Téléchargement.
