Welcome to Pol's web site!

Cool place for free MacOS source code!

This web site offers lots of sample code I've written the past years that covers many areas: Graphic, Sound, User interface, Files... Although it's not documented, this source code is quite clean so you shouldn't have any problem including it in your code.

All the code available on this web site is available free of charge and with unrestricted use - even for commercial applications.

Note that some sample code actually includes code from Apple's own sample code.
