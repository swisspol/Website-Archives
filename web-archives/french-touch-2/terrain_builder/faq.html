General questions :

1) What's the purpose of WaterRace Terrain Builder ?

WaterRace Terrain Builder is a terrain editor for WaterRace to be used in conjunction with Fusion, the object editor. With these free tools, players should be able to create totally new levels!

 

2) How much does it cost ?

WaterRace Terrain Builder is free! You may download the WaterRace Terrain Builder package right now by clicking on the download link in the upper left corner of this page and start using it immediately.

 

3) How can I learn to use it ?

WaterRace Terrain Builder is provided with a step-by-step tutorial which includes plenty of screenshots and detailed explanations.

 

4) Is there any difference between the version available on this web site and the version French Touch is using to create the WaterRace levels ?

None: we are using exactly the same version of WaterRace Terrain Builder .

 

5) Can I create terrains with WaterRace Terrain Builder and sell them as extensions for your games ?

So far, the answer is No. If you create something with WaterRace Terrain Builder, it has to be distributed for free.

 

Product Features :

1) Can WaterRace Terrain Builder import terrain meshes created in another software, e.g. Bryce ?

So far no. WaterRace Terrain Builder creates terrain meshes from level maps which are basically gray levels PICT files of any size - well, provided they are not too big.

 

2) Can WaterRace Terrain Builder export the terrain as a standard 3D file ?

No.

 

3) Does WaterRace Terrain Builder support 3Dfx cards ?

No. Unfortunately, most 3Dfx cards are not compatible with WaterRace Terrain Builder. Only 3Dfx cards which are able to render in a window may be able to work with WaterRace Terrain Builder.
