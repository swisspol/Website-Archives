Fusion gallery :

These are a few screenshots of Fusion that show you some features of this powerful tool.
Click on a thumbnail to see it full screen.

 

 

 

Fusion 3.1 redesigned interface.

 

 

 

 

 

 

 

    Fusion 3.0 introduces skeleton animation which allows to create truly complex animations in real-time even with shapes with thousands of points!

 

 

 

 

 



The brand new interface of Fusion 2.0 in which the inspector single window (here on the right) replaces most of the floating palettes.
Fusion 2.5 adds mouse navigation in the views, shape selection by the mouse and contextual menus.

 

 

 

 

 

    Fusion 2.5 has many improvements related to character animation: rotoscoping, world box...

 

 

 

 

 

Fusion allows you to edit a texture mapping vertex per vertex.

 

 

 

 

 

 

    Fusion allows complex linking between shaped - up to 4 levels depth.

 

 

 

 

 

 

 Fusion has a powerful onboard sequencer which allows you to write complex animation scripts using different kinds of interpolation (linear and quaternions).

 

 

 

 

 

 

    Using the Fusion sequencer, you can even write movies: build the scene, place the actors, place the camera and "action"!

 

 

 

 

 

With Fusion 2.0, you can store different texture sets for the same object in a single object file.
