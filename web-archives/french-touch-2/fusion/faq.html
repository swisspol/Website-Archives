General questions :

1) What's the purpose of Fusion ?

Fusion is an object editor. There is also a terrain editor for WaterRace (WaterRace Terrain Builder). With these free tools, players should be able to create totally new levels!

 

2) How much does it cost ?

Fusion is free! You may download the Fusion package right now by clicking on the download link in the upper left corner of this page and start using it immediately.

 

3) How can I learn to use it ?

Fusion is provided with a very detailed 50 pages manual in PDF format and many example files.

 

4) Is there any difference between the version available on this web site and the version you're using to create the models ?

We are using exactly the same version of Fusion.

 

5) Can I create models in Fusion and sell them as extensions for your games ?

So far, the answer is No. If you create something with Fusion, it has to be distributed for free.

 

Product Features :

1) What kind of 3D files can Fusion import ?

Fusion can import the mesh and trimesh data contained in 3DMF (QuickDraw 3D file format) files.

 

2) What kind of 3D files can Fusion export to ?

Currently, there's only one: Infinity 3D object file format, but we might consider implementing a 3DMF export option.

 

3) Does Fusion support 3Dfx cards ?

No. Unfortunately, most 3Dfx cards are not compatible with Fusion. Only 3Dfx which are able to render in a window may be able to work with Fusion.
