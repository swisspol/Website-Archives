General :

1) When will WaterRace be out ?

Sorry, since we're continuously adding new features to the game, we won't give any release date until we're almost over.

 

2) How will it be distributed ?

At the beginning, we planned to distribute WaterRace as a shareware over the Internet. That's why we first designed the game so it could fit in a few Mb archive. Finally, the game is going to be much bigger than what we expected; and in order to preserve graphic and audio quality, the game will be available on CD only.
Of course if we find a publisher for the game we might consider going commercial.

 

3) Will I be able to create my own boats or circuits ?

Sure, thanks to Fusion and WaterRace Terrain Builder. WaterRace is the most open game ever made! ;-)

 

4) What does DR stand for? ?

DR means "Developer Release" i.e. this version of the game is for internal use only. On the contrary, the suffix "Beta" means this version will be sent to our beta-testers.

 

Performances :

1) What are the actual performances of the game ?

The WaterRace Preview Edition runs between 20 and 30 FPS in 640x480 millions of color on an IMac-B (with Rage Pro 6Mb) with full visual effects.

 

2) What is the required configuration to run WaterRace?

Any G3 computer or PowerMacintosh PCI with Rave compliant 3D acceleration and more than 2Mb of Video Memory (VRAM).
If necessary, we might develop a software-only version of Infinity so the game can run on any PowerMac.
