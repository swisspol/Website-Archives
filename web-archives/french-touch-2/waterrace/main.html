Description

WaterRace is the first boat racing game available on the Power Mac platform.

The players will be able to face opponents either simulated by the computer, or in network play.

WaterRace brings in addition many innovations to this concept: full 3D, really open architecture of the game, many "vehicles" and animated terrains...

WaterRace uses a special version of the 3D engine developed by French Touch which supports special effects like Soft Shadows, Lens Flares...

The open architecture of the program will make it possible for any player to easily add extension packs (new boats or new terrains). More information regarding the creation of extension packs and a list of the best ones is available on the mapmakers page.

All the terrains will be different not only by their difficulty but also by their "atmosphere" so... special:Venice, Jamaica, Antartic...

Such features promise an almost unlimited lifetime for the game!

 

Features

It's the first game completely designed for Apple's new G3 computers. This means:

  • exclusive 100% Rave 3D engine (Infinity) that communicates directly with the ATI or 3Dfx 3D chips for awesome performances
  • high resolution 3D models
  • interface and game in thousands of colors and animated in real-time 3D
  • visual effects like lens flares and soft shadows
  • CD quality sound effects and music

WaterRace will also support network games over Local Area Network and maybe over the Internet.

 

Software and hardware requirements

WaterRace runs on any PowerMacintosh PCI with Rave compliant 3D acceleration, but for the best performances and rendering quality, a G3 is highly recommended (see the Infinity page for more information about this).
