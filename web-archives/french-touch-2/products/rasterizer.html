Rasterizer

 

 

What is Rasterizer :

Rasterizer is utility that renders hi-resolution screenshots of Fusion objects (up to 16000x16000 pixels provided Rasterizer has enough free memory).

 

System requirements :

To be able to use Rasterizer, you need:
• a PCI PowerMacintosh - a G3 (all G3 Power Macintoshes have ATI 3D chips on board) or a PCI PowerMacintosh with a 3D accelerator card is highly recommended.
• System 7.1 or later.
• QuickDraw 3D Rave.
• the Appearance Manager.

 

Version history :

1.1 Support for Fusion 3.1 file format.

1.0 First public release.
