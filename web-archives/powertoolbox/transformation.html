What is the Transformation Library:

This library allows you to apply some transformations on PixMaps:
  • translation
  • scaling (horizontal & vertical)
  • rotation


Main caracteristics:

  • Supports 8 Bits, 16 Bits and 32 Bits depths.
  • Uses matrix operations.
  • All calculations are done in floating point for accurate precision.

System requirements:
  • PowerPC processor MacOS system.
  • System 7.1 or later.
