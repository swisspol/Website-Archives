WaterRace free add-ons collection

To install these add-ons for WaterRace, simply uncompress them and drop them into the "Add-ons" folder of the "WaterRace Data" folder, then launch the game and create a new pilot in "Add-on" mode.
If you have created add-ons using the editing tools provided with the game, send them to us so that we can host them on the server.
