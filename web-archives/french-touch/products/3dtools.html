4 Tools available

Here's a list of our free 3D tools based on the Infinity 3D engine.
These tools allow you to transform your creations made with Infinity Model Builder to stand-alone animations, QuickTime™ movies, render them directly in Adobe Photoshop™... and even use them in your program if you are a developer.

Cinema

Cinema is a tool that convert 3D model animations stored in Infinity MetaFiles into standard QuickTime™ movies. It is less than 150Kb but includes a complete version of the Infinity 3D Engine.

Create awesome 3D real-time animation using Infinity Model Builder, then render them into QuickTime™ movies with Cinema!

To be able to use Cinema, you need:
• a PowerMacintosh - a G3 (all G3 Power Macintoshes have ATI 3D chips on board) or a PowerMacintosh with a 3D accelerator card is highly recommended.
• System 7.6 or later.
• QuickDraw 3D Rave.

Download Cinema

D-Fusion

D-Fusion is a stand alone player for 3D model animations stored in Infinity MetaFiles. It is less than 150Kb but includes a complete version of the Infinity 3D Engine, allowing you to play full screen the animations contained in any Infinity MetaFile by just dropping them onto D-Fusion's icon or double-clicking onto the D-Fusion's icon.

Create awesome 3D real-time animation using Infinity Model Builder, then just distribute them over the Internet with the D-Fusion player!

To be able to use D-Fusion, you need:
• a PowerMacintosh - a G3 (all G3 Power Macintoshes have ATI 3D chips on board) or a PowerMacintosh with a 3D accelerator card is highly recommended.
• System 7.6 or later.
• QuickDraw 3D Rave.

Player SDK

The Infinity Player SDK allows any developer to use the power of the Infinity 3D Engine: you can now play in any application real-time 3D animations.

The "InfinityPlayerLib" library: allows you to play any animation script embended in a MetaFile. It provides the following possibilities:
• can use a shape as a fake camera
• custom setup of the lights
• custom setup of the camera position
• custom setup of the clipping distance (hither and yon distances)
• start playing another animation while one is already playing
• can render onto an image background
• true aliased or low-resolution rendering on ATI hardware
• lighting FX
• particle systems
• custom audio engine

To be able to use the Player SDK, you need:
• a PowerMacintosh - a G3 (all G3 Power Macintoshes have ATI 3D chips on board) or a PowerMacintosh with a 3D accelerator card is highly recommended.
• System 7.6 or later.
• QuickDraw 3D Rave.
• Photoshopª 5.0 or later

Rasterizer

Rasterizer is a utility that renders hi-resolution screenshots of 3D models included in Infinity MetaFiles (up to 16000x16000 pixels provided Rasterizer has enough free memory).

Rasterizer Plug is a plug-in for Photoshop™ 5.0 or later that renders hi-resolution screenshots of 3D models included in Infinity MetaFiles directly onto a Photoshop™ layer with transparency.

To be able to use Rasterizer, you need:
• a PowerMacintosh - a G3 (all G3 Power Macintoshes have ATI 3D chips on board) or a PowerMacintosh with a 3D accelerator card is highly recommended.
• System 7.6 or later.
• QuickDraw 3D Rave.
• Photoshopª 5.0 or later
