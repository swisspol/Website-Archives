3 Tools available

Here's a list of our free 3D game creation and editing tools, all of them with real-time rendering through the Infinity 3D engine.
These are simply the fastest and easiest tools ever made!

The tools share a unique common file format with the game files: the Infinity MetaFile format. This allows you to modify and create directly 3D models and terrains for our games.

Powerful features of the Infinity Engine include:

• high-precision (1/100 of second) sequencer with linear or quaternion interpolation,
• linking,
• skeleton-based mesh deformation,
• hi-quality QuickTime™ compressed textures,
• texture transparency,
• vertex-based transparency,
• Gouraud, Lambert and pseudo-Phong shading,
• special FX and particle systems,
• aliasing & low resolution rendering,
• rendering directly onto the screen e.g. render a 3D character directly into the Finder's desktop!
• support for multiple audio tracks,
• and much more...

Infinity Model Builder

Model Builder is a free powerful 3D editor designed to create and edit object files with real-time rendering through the Infinity 3D Rave engine. Model Builder can import the meshes contained in 3DMF files, DXF files or OBJ files, modify them (position, rotation, scale...), add colors or textures, and save the result as a standard Infinity MetaFile.

Model Builder was developed totally from the ground up as a real-time editor and is based on the Infinity 3D engine. As a consequence, it is the fastest 3D tool available on the Mac platform even on low-end PowerMacs.

Model Builder is not a 3D modeler: it does not allow you to create meshes.
Model Builder allows you to edit and modify any 3D model included with the game WaterRace.
You can also use Model Builder to create 3D models you can include in the terrains you build with Infinity Terrain Builder, the 3D terrain editor available for free from French Touch.

To be able to use Model Builder, you need:
• a PowerMacintosh - a G3 (all G3 Power Macintoshes have ATI 3D chips on board) or a PowerMacintosh with a 3D accelerator card is highly recommended.
• System 7.6 or later.
• QuickDraw 3D Rave.

Infinity Terrain Builder

Terrain Builder is a powerful 3D world editor designed to create and edit levels (or terrains) for French Touch games with real-time rendering through the Infinity 3D engine.

You can use to modify current WaterRace levels or create totally amazing levels from the ground up.

Terrain Builder is the best 3D level editing tool for the Mac.

Terrain Builder creates new terrains using gray levels images. You can then apply textures and lighting on these terrains, add objects, place cameras, create animations… and most of all, Terrain Builder is easy to use.

To be able to use Terrain Builder, you need:
• a PowerMacintosh - a G3 (all G3 Power Macintoshes have ATI 3D chips on board) or a PowerMacintosh with a 3D accelerator card is highly recommended.
• System 7.6 or later.
• QuickDraw 3D Rave.
